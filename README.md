# 🎤 Spiel — Conversations, Simplified

🔗 **Live Demo:** https://spiel-voice.vercel.app/

---

##  Overview

Spiel is a thoughtfully designed web application that generates dynamic speech and discussion topics to help users practice articulation, stimulate ideas, and improve communication skills.

The application emphasizes simplicity, responsiveness, and a refined user experience.

---

##  Key Features

* 🎯 **Dynamic Topic Generation** — Instantly generates engaging discussion prompts
* 🧠 **Category-Based Filtering** — Explore topics across Tech, Career, and Fun
* ⚡ **Seamless Interaction** — Fast and responsive UI updates
* 🎨 **Modern Interface** — Clean, minimal dark-themed design
* ✨ **Smooth Animations** — Subtle transitions for enhanced experience
* 📱 **Responsive Layout** — Optimized for different screen sizes
